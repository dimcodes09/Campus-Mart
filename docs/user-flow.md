# User Flow

## 1. Onboarding
- User signs up
- Verifies identity

## 2. Product Listing (Seller)
- Upload product details
- Set rent/price/deposit
- Publish listing

## 3. Discovery (Buyer)
- Browse products
- Use search/filter
- View recommendations

## 4. Interaction
- Open product
- Chat with seller

## 5. Rental Flow
- Select duration
- Confirm rental
- Agreement generated

## 6. Pickup
- QR scan (optional)
- Status becomes active

## 7. Usage Period
- Timer active
- Notifications sent

## 8. Return
- Item returned
- Status completed

## 9. Rating
- Both users rate each other
- Trust score updated
