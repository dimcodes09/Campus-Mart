# Product Requirement Document (PRD)

## Problem Statement
Hostel students frequently need temporary access to items (coolers, books, lab tools), but lack a trusted and structured platform.

## Solution
A verified student marketplace enabling:
- Buying
- Selling
- Renting

with built-in trust and communication.

---

## Target Users
- Hostel students
- College communities

---

## Core Features

1. Authentication with student verification
2. Product listing (buy/sell/rent)
3. Rental system with duration + deposit
4. Real-time chat system
5. AI-based recommendations
6. Trust system (ratings, reliability)

---

## Unique Value Proposition
- Verified users only
- Rental-focused ecosystem
- Trust layer with tracking and ratings

---

## Success Metrics
- Number of listings
- Rental completion rate
- User engagement
- Repeat usage
